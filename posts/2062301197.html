1гҖҒжҰӮеҝөпјҡВ·

  • и·Ёз«ҷи„ҡжң¬ж”»еҮ»пјҢж”»еҮ»иҖ…еңЁзҪ‘з«ҷжіЁе…ҘжҒ¶ж„Ҹи„ҡжң¬пјҢеҪ“з”ЁжҲ·жөҸи§ҲзҪ‘йЎөж—¶пјҢеөҢе…ҘзҪ‘йЎөзҡ„Scriptд»Јз Ғдјҡжү§иЎҢпјҢйҖҡиҝҮеҜ№з”ЁжҲ·жөҸи§ҲеҷЁиҝӣиЎҢжҺ§еҲ¶жҲ–иҖ…жӢҝеҲ°з”ЁжҲ·йҡҗз§Ғж•°жҚ®
  • йҮҚзӮ№дёҚеңЁдәҺи·Ёз«ҷзӮ№пјҢXSS зҡ„жң¬иҙЁжҳҜеӣ дёәзҪ‘з«ҷжІЎжңүеҜ№жҒ¶ж„Ҹд»Јз ҒиҝӣиЎҢиҝҮж»ӨпјҢдёҺжӯЈеёёзҡ„д»Јз Ғж··еҗҲеңЁдёҖиө·дәҶпјҢжөҸи§ҲеҷЁжІЎжңүеҠһжі•еҲҶиҫЁе“Әдәӣи„ҡжң¬жҳҜеҸҜдҝЎзҡ„пјҢд»ҺиҖҢеҜјиҮҙдәҶжҒ¶ж„Ҹд»Јз Ғзҡ„жү§иЎҢгҖӮ
  • ж”»еҮ»иҖ…еҸҜд»ҘйҖҡиҝҮиҝҷз§Қж”»еҮ»ж–№ејҸеҸҜд»ҘиҝӣиЎҢд»ҘдёӢж“ҚдҪңпјҡ
    • иҺ·еҸ–йЎөйқўзҡ„ж•°жҚ®пјҢеҰӮDOMгҖҒcookieгҖҒlocalStorageпјӣ
    • DOSж”»еҮ»пјҢеҸ‘йҖҒеҗҲзҗҶиҜ·жұӮпјҢеҚ з”ЁжңҚеҠЎеҷЁиө„жәҗпјҢд»ҺиҖҢдҪҝз”ЁжҲ·ж— жі•и®ҝй—®жңҚеҠЎеҷЁпјӣ
    • з ҙеқҸйЎөйқўз»“жһ„пјӣ
    • жөҒйҮҸеҠ«жҢҒпјҲе°Ҷй“ҫжҺҘжҢҮеҗ‘жҹҗзҪ‘з«ҷпјүпјӣ

2гҖҒзұ»еһӢВ·

1гҖҒеҸҚе°„еһӢ/йқһжҢҒд№…еһӢВ·
  • ж”»еҮ»иҖ…иҜұеҜјз”ЁжҲ·и®ҝй—®дёҖдёӘеёҰжңүжҒ¶ж„Ҹд»Јз Ғзҡ„ URL еҗҺпјҢжңҚеҠЎеҷЁз«ҜжҺҘж”¶ж•°жҚ®еҗҺеӨ„зҗҶпјҢз„¶еҗҺжҠҠеёҰжңүжҒ¶ж„Ҹд»Јз Ғзҡ„ж•°жҚ®еҸ‘йҖҒеҲ°жөҸи§ҲеҷЁз«ҜпјҢжөҸи§ҲеҷЁз«Ҝи§Јжһҗиҝҷж®өеёҰжңү XSS д»Јз Ғзҡ„ж•°жҚ®еҗҺеҪ“еҒҡи„ҡжң¬жү§иЎҢпјҢжңҖз»Ҳе®ҢжҲҗ XSS ж”»еҮ»гҖӮ йңҖиҰҒжңҚеҠЎз«ҜдәӨдә’
  • еҸҚе°„еһӢ XSS зҡ„ж”»еҮ»жӯҘйӘӨпјҡ
    1. ж”»еҮ»иҖ…жһ„йҖ еҮәзү№ж®Ҡзҡ„ URLпјҢе…¶дёӯеҢ…еҗ«жҒ¶ж„Ҹд»Јз ҒгҖӮ
    2. вҪӨжҲ·жү“ејҖеёҰжңүжҒ¶ж„Ҹд»Јз Ғзҡ„ URL ж—¶пјҢвҪ№з«ҷжңҚеҠЎз«Ҝе°ҶжҒ¶ж„Ҹд»Јз Ғд»Һ URL дёӯеҸ–еҮәпјҢжӢјжҺҘеңЁ HTML дёӯиҝ”еӣһз»ҷжөҸи§ҲеҷЁгҖӮ
    3. вҪӨжҲ·жөҸи§ҲеҷЁжҺҘж”¶еҲ°е“Қеә”еҗҺи§Јжһҗжү§вҫҸпјҢж··еңЁе…¶дёӯзҡ„жҒ¶ж„Ҹд»Јз Ғд№ҹиў«жү§вҫҸгҖӮ
    4. жҒ¶ж„Ҹд»Јз ҒзӘғеҸ–вҪӨжҲ·ж•°жҚ®е№¶еҸ‘йҖҒеҲ°ж”»еҮ»иҖ…зҡ„вҪ№з«ҷпјҢжҲ–иҖ…еҶ’е……вҪӨжҲ·зҡ„вҫҸдёәпјҢи°ғвҪӨвҪ¬ж ҮвҪ№з«ҷжҺҘвјқжү§вҫҸж”»еҮ»иҖ…жҢҮе®ҡзҡ„ж“ҚдҪңгҖӮ
  • еңәжҷҜпјҡйҖҡиҝҮ URL дј йҖ’еҸӮж•°зҡ„еҠҹиғҪпјҢеҰӮзҪ‘з«ҷжҗңзҙўгҖҒи·іиҪ¬зӯүгҖӮз”ұдәҺйңҖиҰҒз”ЁжҲ·дё»еҠЁжү“ејҖжҒ¶ж„Ҹзҡ„ URL жүҚиғҪз”ҹж•ҲпјҢж”»еҮ»иҖ…еҫҖеҫҖдјҡз»“еҗҲеӨҡз§ҚжүӢж®өиҜұеҜјз”ЁжҲ·зӮ№еҮ»гҖӮ
2гҖҒеӯҳеӮЁеһӢ/жҢҒд№…еһӢВ·

  • жҠҠжҒ¶ж„Ҹи„ҡжң¬еӯҳеӮЁеҲ°жңҚеҠЎеҷЁзҡ„ж•°жҚ®еә“еҪ“дёӯпјҢжөҸи§ҲеҷЁиҜ·жұӮж•°жҚ®ж—¶пјҢи„ҡжң¬е°ұиў«дј еӣһе№¶жү§иЎҢпјҢиҫҫеҲ°ж”»еҮ»ж•Ҳжһң

  • еӯҳеӮЁеһӢ XSS зҡ„ж”»еҮ»жӯҘйӘӨпјҡ

    1. ж”»еҮ»иҖ…е°ҶжҒ¶ж„Ҹд»Јз ҒжҸҗдәӨеҲ°вҪ¬ж ҮвҪ№з«ҷзҡ„ж•°жҚ®еә“дёӯгҖӮ
    2. вҪӨжҲ·жү“ејҖвҪ¬ж ҮвҪ№з«ҷж—¶пјҢвҪ№з«ҷжңҚеҠЎз«Ҝе°ҶжҒ¶ж„Ҹд»Јз Ғд»Һж•°жҚ®еә“еҸ–еҮәпјҢжӢјжҺҘеңЁ HTML дёӯиҝ”еӣһз»ҷжөҸи§ҲеҷЁгҖӮ
    3. вҪӨжҲ·жөҸи§ҲеҷЁжҺҘж”¶еҲ°е“Қеә”еҗҺи§Јжһҗжү§вҫҸпјҢж··еңЁе…¶дёӯзҡ„жҒ¶ж„Ҹд»Јз Ғд№ҹиў«жү§вҫҸгҖӮ
    4. жҒ¶ж„Ҹд»Јз ҒзӘғеҸ–вҪӨжҲ·ж•°жҚ®е№¶еҸ‘йҖҒеҲ°ж”»еҮ»иҖ…зҡ„вҪ№з«ҷпјҢжҲ–иҖ…еҶ’е……вҪӨжҲ·зҡ„вҫҸдёәпјҢи°ғвҪӨвҪ¬ж ҮвҪ№з«ҷжҺҘвјқжү§вҫҸж”»еҮ»иҖ…жҢҮе®ҡзҡ„ж“ҚдҪңгҖӮ

  • дёҫдҫӢпјҡ

    еҪ“жңүдәәеңЁз•ҷиЁҖеҶ…е®№дёӯжҸ’е…ҘжҒ¶ж„Ҹи„ҡжң¬ж—¶пјҢз•ҷиЁҖиӮҜе®ҡдјҡдҝқеӯҳеҲ°жңҚеҠЎеҷЁдёӯгҖӮ

    з”ұдәҺжңҚеҠЎеҷЁиҰҒеғҸжҜҸдёҖдёӘи®ҝе®ўеұ•зӨәд№ӢеүҚзҡ„з•ҷиЁҖеҶ…е®№пјҢжүҖд»ҘеҗҺйқўзҡ„и®ҝе®ўиҮӘз„¶дјҡжҺҘж”¶еҲ°д№ӢеүҚз•ҷиЁҖдёӯзҡ„жҒ¶ж„Ҹи„ҡжң¬иҖҢдёҚе№ёиәәжһӘ

  • еңәжҷҜпјҡеёҰжңүз”ЁжҲ·дҝқеӯҳж•°жҚ®зҡ„зҪ‘з«ҷеҠҹиғҪпјҢжҜ”еҰӮи®әеқӣеҸ‘её–гҖҒе•Ҷе“ҒиҜ„д»·гҖҒз”ЁжҲ·з§ҒдҝЎзӯүзӯү

  • еҸҚе°„еһӢ XSS и·ҹеӯҳеӮЁеһӢ XSS зҡ„еҢәеҲ«жҳҜпјҡ

    еӯҳеӮЁеһӢ XSS зҡ„жҒ¶ж„Ҹд»Јз ҒеӯҳеңЁж•°жҚ®еә“вҫҘпјҢеҸҚе°„еһӢ XSS зҡ„жҒ¶ж„Ҹд»Јз ҒеӯҳеңЁ URL вҫҘ

3гҖҒDOMеһӢВ·
  • дёҚйңҖиҰҒжңҚеҠЎз«ҜдәӨдә’пјҢзәҜзІ№еҸ‘з”ҹеңЁе®ўжҲ·з«Ҝ
  • йҖҡиҝҮжҒ¶ж„Ҹи„ҡжң¬дҝ®ж”№DOMз»“жһ„пјҢйҖҡиҝҮдҝ®ж”№йЎөйқўзҡ„ DOM иҠӮзӮ№еҪўжҲҗзҡ„ XSSгҖӮ
  • DOM еһӢ XSS зҡ„ж”»еҮ»жӯҘйӘӨпјҡ
    1. ж”»еҮ»иҖ…жһ„йҖ еҮәзү№ж®Ҡзҡ„ URLпјҢе…¶дёӯеҢ…еҗ«жҒ¶ж„Ҹд»Јз ҒгҖӮ
    2. вҪӨжҲ·жү“ејҖеёҰжңүжҒ¶ж„Ҹд»Јз Ғзҡ„ URLгҖӮ
    3. вҪӨжҲ·жөҸи§ҲеҷЁжҺҘж”¶еҲ°е“Қеә”еҗҺи§Јжһҗжү§вҫҸпјҢеүҚз«Ҝ JavaScript еҸ–еҮә URL дёӯзҡ„жҒ¶ж„Ҹд»Јз Ғе№¶жү§вҫҸгҖӮ
    4. жҒ¶ж„Ҹд»Јз ҒзӘғеҸ–вҪӨжҲ·ж•°жҚ®е№¶еҸ‘йҖҒеҲ°ж”»еҮ»иҖ…зҡ„вҪ№з«ҷпјҢжҲ–иҖ…еҶ’е……вҪӨжҲ·зҡ„вҫҸдёәпјҢи°ғвҪӨвҪ¬ж ҮвҪ№з«ҷжҺҘвјқжү§вҫҸж”»еҮ»иҖ…жҢҮе®ҡзҡ„ж“ҚдҪңгҖӮ
  • дёҫдҫӢпјҡзңӢе°ҸиҜҙзҡ„ж—¶еҖҷдёӢйқўжңүе°Ҹе№ҝе‘ҠпјҢзӣҙжҺҘж”№еҸҳеҪ“еүҚйЎөйқўзҡ„Domз»“жһ„
  • DOM еһӢ XSS и·ҹеүҚдёӨз§Қ XSS зҡ„еҢәеҲ«пјҡ
    • DOM еһӢ XSS ж”»еҮ»дёӯпјҢеҸ–еҮәе’Ңжү§вҫҸжҒ¶ж„Ҹд»Јз Ғз”ұжөҸи§ҲеҷЁз«Ҝе®ҢжҲҗпјҢеұһдәҺеүҚз«ҜJavaScript вҫғиә«зҡ„е®үе…ЁжјҸжҙһпјҢвҪҪе…¶д»–дёӨз§Қ XSS йғҪеұһдәҺжңҚеҠЎз«Ҝзҡ„е®үе…ЁжјҸжҙһгҖӮ

3гҖҒйҳІиҢғжҺӘж–ҪпјҡВ·

ж–№ејҸ1пјҡд»ҺжөҸи§ҲеҷЁзҡ„жү§иЎҢжқҘиҝӣиЎҢйў„йҳІпјҢдҪҝз”ЁзәҜеүҚз«Ҝзҡ„ж–№ејҸпјҡВ·
  1. дҪҝз”ЁзәҜеүҚз«Ҝзҡ„ж–№ејҸпјҢдёҚз”ЁжңҚеҠЎеҷЁз«ҜжӢјжҺҘеҗҺиҝ”еӣһпјҲдёҚдҪҝз”ЁжңҚеҠЎз«ҜжёІжҹ“пјү
  2. еҜ№йңҖиҰҒжҸ’е…ҘеҲ° HTML дёӯзҡ„д»Јз ҒеҒҡеҘҪе……еҲҶзҡ„иҪ¬д№үгҖӮеҜ№дәҺ DOM еһӢзҡ„ж”»еҮ»пјҢдё»иҰҒжҳҜеүҚз«Ҝи„ҡжң¬зҡ„дёҚеҸҜйқ иҖҢйҖ жҲҗзҡ„пјҢеҜ№дәҺж•°жҚ®иҺ·еҸ–жёІжҹ“е’Ңеӯ—з¬ҰдёІжӢјжҺҘзҡ„ж—¶еҖҷеә”иҜҘеҜ№еҸҜиғҪеҮәзҺ°зҡ„жҒ¶ж„Ҹд»Јз Ғжғ…еҶөиҝӣиЎҢеҲӨж–ӯгҖӮ
    • жӢ’з»қinnerHTML, дҪҝз”Ё.innerTextгҖҒ.setAttributeгҖҒ.styleпјҢ
    • VueпјҡдҪ зҡ„з«ҷзӮ№дёҠеҠЁжҖҒжёІжҹ“зҡ„д»»ж„Ҹ HTML еҸҜиғҪдјҡйқһеёёеҚұйҷ©пјҢеӣ дёәе®ғеҫҲе®№жҳ“еҜјиҮҙ XSS ж”»еҮ»гҖӮиҜ·еҸӘеҜ№еҸҜдҝЎеҶ…е®№дҪҝз”Ё HTML жҸ’еҖјпјҢз»қдёҚиҰҒеҜ№з”ЁжҲ·жҸҗдҫӣзҡ„еҶ…е®№дҪҝз”ЁжҸ’еҖј
ж–№ејҸ2пјҡдҪҝз”ЁCSPзҷҪеҗҚеҚ•В·
  • CSP зҡ„жң¬иҙЁжҳҜе»әз«ӢдёҖдёӘзҷҪеҗҚеҚ•пјҢе‘ҠиҜүжөҸи§ҲеҷЁе“ӘдәӣеӨ–йғЁиө„жәҗеҸҜд»ҘеҠ иҪҪе’Ңжү§иЎҢпјҢд»ҺиҖҢйҳІжӯўжҒ¶ж„Ҹд»Јз Ғзҡ„жіЁе…Ҙж”»еҮ»гҖӮ
  • еҜ№дәҺз”ЁжҲ·зҡ„д»»дҪ•иҫ“е…ҘиҰҒиҝӣиЎҢиҪ¬з Ғе’ҢиҝҮж»ӨгҖӮе»әз«ӢеҸҜдҝЎд»»зҡ„еӯ—з¬Ұе’Ң HTML ж ҮзӯҫзҷҪеҗҚеҚ•пјҢеҜ№дәҺдёҚеңЁзҷҪеҗҚеҚ•д№ӢеҲ—зҡ„еӯ—з¬ҰжҲ–иҖ…ж ҮзӯҫиҝӣиЎҢиҝҮж»ӨжҲ–зј–з ҒгҖӮ
    1. CSP жҢҮзҡ„жҳҜеҶ…е®№е®үе…Ёзӯ–з•ҘпјҢе®ғзҡ„жң¬иҙЁжҳҜе»әз«ӢдёҖдёӘзҷҪеҗҚеҚ•пјҢе‘ҠиҜүжөҸи§ҲеҷЁе“ӘдәӣеӨ–йғЁиө„жәҗеҸҜд»ҘеҠ иҪҪе’Ңжү§иЎҢгҖӮжҲ‘д»¬еҸӘйңҖиҰҒй…ҚзҪ®и§„еҲҷпјҢеҰӮдҪ•жӢҰжҲӘз”ұжөҸи§ҲеҷЁиҮӘе·ұжқҘе®һзҺ°гҖӮ
    2. йҖҡеёёжңүдёӨз§Қж–№ејҸжқҘејҖеҗҜ CSPпјҢдёҖз§ҚжҳҜи®ҫзҪ® HTTP йҰ–йғЁдёӯзҡ„ Content-Security-PolicyпјҢдёҖз§ҚжҳҜи®ҫзҪ® meta ж Үзӯҫзҡ„ж–№ејҸ
ж–№ејҸ3пјҡcookieе®үе…Ёзӯ–з•ҘпјҡВ·

  • еҫҲеӨҡxssж”»еҮ»йғҪжҳҜдёәиҺ·еҸ–cookieдёӯеҶ…е®№пјҢиҝҷж ·жөҸи§ҲеҷЁзҰҒжӯўJSи®ҝй—®еёҰжңүHTTPOnlyеұһжҖ§зҡ„cookieпјҢд№ҹеҸҜд»ҘдҪҝз”ЁйӘҢиҜҒз ҒпјҢйҒҝе…Қи„ҡжң¬дјӘиЈ…жҲҗз”ЁжҲ·жү§иЎҢдёҖдәӣж“ҚдҪңгҖӮ

  • дҪҝз”ЁеңәжҷҜпјҡ

    жҲ‘д»¬и®ҝй—®зҪ‘з«ҷзҡ„ж—¶еҖҷиҙҰжҲ·дҝЎжҒҜеҸҜиғҪеӯҳеӮЁеңЁcookieдёӯпјҢж–№дҫҝз”ЁжҲ·йӘҢиҜҒдҝЎжҒҜпјҢиҪ¬иҙҰзӯүйЎөйқўж“ҚдҪңгҖӮиҝҷз§Қж“ҚдҪңеҸҜиғҪеј•еҸ‘XSSж”»еҮ»пјҲеҸҚе°„еһӢпјүпјҢйҖҡиҝҮжү§иЎҢеҪ“еүҚйЎөйқўзҡ„JSд»Јз ҒпјҢиҺ·еҸ–еҲ°cookieйҮҢдҝЎжҒҜпјҢеҸ‘з»ҷж”»еҮ»иҖ…зҡ„зҪ‘з«ҷ

  • **еҒҡжі•пјҡ**еңЁжңҚеҠЎз«Ҝи®ҫзҪ®йҰ–йғЁеӯ—ж®өset-cookieдёәHTTPOnlyпјҢиҝҷж ·еҸҜд»ҘйҳІжӯўз”ЁжҲ·йҖҡиҝҮJSиҺ·еҸ–cookieгҖӮеҜ№cookieзҡ„иҜ»еҶҷ/еҸ‘йҖҒдёҖиҲ¬жңүеҰӮдёӢеӯ—ж®өи®ҫзҪ®пјҡ

    • HTTP-OnlyпјҡеҸӘе…Ғи®ёHTTPS/httpиҜ·жұӮиҜ»еҸ–cookieпјҢJSд»Јз ҒжҳҜж— жі•иҜ»еҸ–cookieзҡ„пјҢеҸ‘йҖҒиҜ·жұӮж—¶иҮӘеҠЁеҸ‘йҖҒcookie
    • secure-OnlyпјҡеҸӘе…Ғи®ёHTTPSиҜ·жұӮиҜ»еҸ–пјҢеҸ‘йҖҒиҜ·жұӮж—¶иҮӘеҠЁеҸ‘йҖҒcookie
    • host-OnlyпјҡеҸӘе…Ғи®ёдё»жңәеҹҹеҗҚдёҺdomainи®ҫзҪ®е®ҢжҲҗдёҖиҮҙзҡ„зҪ‘з«ҷжүҚжүҚиғҪи®ҝй—®cookie

2гҖҒCSRFВ·

1гҖҒжҰӮеҝөВ·

и·Ёз«ҷиҜ·жұӮдјӘйҖ пјҡ

  • ж”»еҮ»иҖ…иҜұеҜјз”ЁжҲ·иҝӣе…ҘдёҖдёӘз¬¬дёүж–№зҪ‘з«ҷпјҢз„¶еҗҺиҜҘзҪ‘з«ҷеҗ‘иў«ж”»еҮ»зҪ‘з«ҷеҸ‘йҖҒи·Ёз«ҷиҜ·жұӮгҖӮ
  • еҰӮжһңз”ЁжҲ·еңЁиў«ж”»еҮ»зҪ‘з«ҷдёӯдҝқеӯҳдәҶзҷ»еҪ•зҠ¶жҖҒпјҢйӮЈд№Ҳж”»еҮ»иҖ…е°ұеҸҜд»ҘеҲ©з”ЁиҝҷдёӘзҷ»еҪ•зҠ¶жҖҒпјҢз»•иҝҮеҗҺеҸ°зҡ„з”ЁжҲ·йӘҢиҜҒпјҢеҶ’е……з”ЁжҲ·еҗ‘жңҚеҠЎеҷЁжү§иЎҢдёҖдәӣж“ҚдҪңгҖӮ
  • CSRF ж”»еҮ»зҡ„жң¬иҙЁжҳҜеҲ©з”Ё cookie дјҡеңЁеҗҢжәҗиҜ·жұӮдёӯжҗәеёҰеҸ‘йҖҒз»ҷжңҚеҠЎеҷЁзҡ„зү№зӮ№пјҢд»ҘжӯӨжқҘе®һзҺ°з”ЁжҲ·зҡ„еҶ’е……гҖӮ

дёҫдҫӢпјҡ

  • жҜ”еҰӮзҷ»еҪ•дёҖдёӘй“¶иЎҢзҡ„зҪ‘йЎөпјҢеңЁжІЎжңүйҖҖеҮәеҪ“еүҚзҪ‘йЎөж—¶пјҲдҝқеӯҳдәҶзҷ»йҷҶжҖҒпјүпјҢдёҖиҲ¬жҳҜй»‘е®ўиҜұдҪҝз”ЁжҲ·зӮ№еҮ»й“ҫжҺҘпјҢжү“ејҖз¬¬дёүж–№зҪ‘з«ҷй»‘е®ўзҪ‘з«ҷдјҡеҸ‘иө·иҜ·жұӮпјҢ
  • з„¶еҗҺеҲ©з”Ёз”ЁжҲ·зҡ„cookieпјҲеҢ…еҗ«з”ЁжҲ·дҝЎжҒҜпјүж¬әйӘ—жңҚеҠЎеҷЁпјҢжңҚеҠЎеҷЁд»ҘдёәиҜ·жұӮж—¶жӯЈеёёз”ЁжҲ·еҸ‘зҡ„е°ұжӯЈеёёеӨ„зҗҶ
  • ж”»еҮ»иҖ…зӣ—з”ЁдәҶз”ЁжҲ·зҡ„иә«д»ҪпјҢд»Ҙз”ЁжҲ·зҡ„еҗҚд№үеҸ‘йҖҒдәҶжҒ¶ж„ҸиҜ·жұӮ

2гҖҒж”»еҮ»зұ»еһӢпјҡВ·

  • getеһӢпјҡжҒ¶ж„ҸзҪ‘з«ҷдјҡиҮӘеҠЁеҸ‘йҖҒhttpиҜ·жұӮпјҢеҗ‘зӣ®ж ҮзҪ‘з«ҷеҸ‘иө·ж”»еҮ»гҖӮжҜ”еҰӮеңЁзҪ‘з«ҷдёӯзҡ„дёҖдёӘ img ж ҮзӯҫйҮҢжһ„е»әдёҖдёӘиҜ·жұӮпјҢеҪ“з”ЁжҲ·жү“ејҖиҝҷдёӘзҪ‘з«ҷзҡ„ж—¶еҖҷе°ұдјҡиҮӘеҠЁеҸ‘иө·жҸҗдәӨгҖӮ
  • postеһӢпјҡдјҡеҲ©з”ЁиЎЁеҚ•жҸҗдәӨпјҢеҗ‘зӣ®ж ҮзҪ‘з«ҷеҸ‘йҖҒpostиҜ·жұӮпјҢиҫҫеҲ°ж”»еҮ»ж•ҲжһңгҖӮжҜ”еҰӮжһ„е»әдёҖдёӘиЎЁеҚ•пјҢз„¶еҗҺйҡҗи—Ҹе®ғпјҢеҪ“з”ЁжҲ·иҝӣе…ҘйЎөйқўж—¶пјҢиҮӘеҠЁжҸҗдәӨиҝҷдёӘиЎЁеҚ•гҖӮ
  • й“ҫжҺҘзұ»еһӢзҡ„ CSRF ж”»еҮ»пјҢжҜ”еҰӮеңЁ a ж Үзӯҫзҡ„ href еұһжҖ§йҮҢжһ„е»әдёҖдёӘиҜ·жұӮпјҢз„¶еҗҺиҜұеҜјз”ЁжҲ·еҺ»зӮ№еҮ»

3гҖҒйҳІиҢғжҺӘж–ҪВ·

1гҖҒиҝӣиЎҢеҗҢжәҗжЈҖжөӢВ·
  • жңҚеҠЎеҷЁж №жҚ® http иҜ·жұӮеӨҙдёӯ origin жҲ–иҖ… referer дҝЎжҒҜжқҘеҲӨж–ӯиҜ·жұӮжҳҜеҗҰдёәе…Ғи®ёи®ҝй—®зҡ„з«ҷзӮ№пјҢд»ҺиҖҢеҜ№иҜ·жұӮиҝӣиЎҢиҝҮж»ӨгҖӮеҪ“ origin жҲ–иҖ… referer дҝЎжҒҜйғҪдёҚеӯҳеңЁзҡ„ж—¶еҖҷпјҢзӣҙжҺҘйҳ»жӯўиҜ·жұӮгҖӮ
    • originпјҡеҢ…еҗ«иҜ·жұӮзҡ„еҹҹеҗҚпјҲдёҚеҢ…еҗ«pathеҸҠqueryпјү
    • referпјҡеҢ…еҗ«е…·дҪ“зҡ„URLең°еқҖ
  • иҝҷз§Қж–№ејҸзҡ„зјәзӮ№жҳҜжңүдәӣжғ…еҶөдёӢ referer еҸҜд»Ҙиў«дјӘйҖ пјҢеҗҢж—¶иҝҳдјҡжҠҠжҗңзҙўеј•ж“Һзҡ„й“ҫжҺҘд№ҹз»ҷеұҸи”ҪдәҶгҖӮжүҖд»ҘдёҖиҲ¬зҪ‘з«ҷдјҡе…Ғи®ёжҗңзҙўеј•ж“Һзҡ„йЎөйқўиҜ·жұӮпјҢдҪҶжҳҜзӣёеә”зҡ„йЎөйқўиҜ·жұӮиҝҷз§ҚиҜ·жұӮж–№ејҸд№ҹеҸҜиғҪиў«ж”»еҮ»иҖ…з»ҷеҲ©з”ЁгҖӮпјҲReferer еӯ—ж®өдјҡе‘ҠиҜүжңҚеҠЎеҷЁиҜҘзҪ‘йЎөжҳҜд»Һе“ӘдёӘйЎөйқўй“ҫжҺҘиҝҮжқҘзҡ„пјү
2гҖҒtokenйӘҢиҜҒВ·

CSRFжҲҗеҠҹзҡ„еҺҹеӣ е°ұжҳҜж”»еҮ»зҡ„зҪ‘з»ңиҜ·жұӮеёҰжңүcookieиҖҢcookieдёӯжңүжӯЈеёёз”ЁжҲ·зҡ„иә«д»ҪдҝЎжҒҜпјҢж¬әйӘ—дәҶжңҚеҠЎеҷЁгҖӮж•…еңЁHTTPиҜ·жұӮдёӯеҠ е…ҘдёҖдёӘйҡҸжңәдә§з”ҹзҡ„tokenпјҢжңҚеҠЎеҷЁжӢҰжҲӘйӘҢиҜҒиҝҷдёӘtokenжҳҜеҗҰеҗҲжі•пјҢдёҚеҗҲжі•е°ұжӢ’з»қиҝҷдёӘзҪ‘з»ңиҜ·жұӮгҖӮ

  • жңҚеҠЎеҷЁйңҖиҰҒз”ҹжҲҗдёҖдёӘйҖҡиҝҮеҠ еҜҶз®—жі•еҠ еҜҶзҡ„ Tokenпјӣ

  • е°ҶиҜҘ Token дҝқеӯҳеңЁ Session жҲ–иҖ… Redisпјӣ

  • е°ҶиҜҘ Token дј з»ҷе®ўжҲ·з«ҜпјҢе®ўжҲ·з«ҜеңЁиҜ·жұӮж—¶иҰҒе°Ҷе…¶ Token дёҖе№¶еёҰдёҠпјҢеҰӮ GET иҜ·жұӮ /api/message?token=xxx жҲ– POST иҜ·жұӮеңЁformиЎЁеҚ•иҝҪеҠ жҲ– Ajax иҜ·жұӮеңЁйЎөйқўж·»еҠ пјҢйҖҡиҝҮ JS еҺ»иҺ·еҸ–иҜҘ meta еҖјпјӣ

  • иҜ·жұӮеҲ°жңҚеҠЎеҷЁеҗҺпјҢжңҚеҠЎеҷЁе°ұеҸҜд»ҘйҖҡиҝҮи§ЈеҜҶеҜ№жҜ”жқҘеҲӨж–ӯиҜҘ Token зҡ„жңүж•ҲжҖ§дәҶгҖӮ

3гҖҒSet-CookieпјҡSameSite вҲҡВ·

йҷҗеҲ¶ cookie дёҚиғҪдҪңдёәиў«з¬¬дёүж–№дҪҝз”ЁпјҢеҚіеңЁе“Қеә”еӨҙ Set-Cookie и®ҫзҪ® SameSite еұһжҖ§пјҢиЎЁзӨәиҜҘ Cookie и®ҝй—®еҗҢжәҗзҪ‘з«ҷиҖҢйқһжқҘжәҗз¬¬дёүж–№зҪ‘з«ҷгҖӮд»ҺиҖҢеҸҜд»ҘйҒҝе…Қиў«ж”»еҮ»иҖ…еҲ©з”ЁгҖӮ

  • Samesite дёҖе…ұжңүдёӨз§ҚжЁЎејҸпјҢдёҖз§ҚжҳҜдёҘж јжЁЎејҸпјҢеңЁдёҘж јжЁЎејҸдёӢ cookie еңЁд»»дҪ•жғ…еҶөдёӢйғҪдёҚеҸҜиғҪдҪңдёәз¬¬дёүж–№ Cookie дҪҝз”ЁпјҢеңЁе®ҪжқҫжЁЎејҸдёӢпјҢcookie еҸҜд»Ҙиў«иҜ·жұӮжҳҜ GET иҜ·жұӮпјҢдё”дјҡеҸ‘з”ҹйЎөйқўи·іиҪ¬зҡ„иҜ·жұӮжүҖдҪҝз”Ё

3гҖҒзҪ‘з»ңеҠ«жҢҒжңүе“ӘеҮ з§ҚпјҢеҰӮдҪ•йҳІиҢғпјҹВ·

вҪ№з»ңеҠ«жҢҒеҲҶдёәдёӨз§Қ:

пјҲ1пјүDNSеҠ«жҢҒ: (иҫ“вјҠдә¬дёңиў«ејәеҲ¶и·іиҪ¬еҲ°ж·ҳе®қиҝҷе°ұеұһдәҺdnsеҠ«жҢҒ)

  • DNSејәеҲ¶и§Јжһҗ: йҖҡиҝҮдҝ®ж”№иҝҗиҗҘе•Ҷзҡ„жң¬ең°DNSи®°еҪ•пјҢжқҘеј•еҜјвҪӨжҲ·жөҒйҮҸеҲ°зј“еӯҳжңҚеҠЎеҷЁ
  • 302и·іиҪ¬зҡ„вҪ…ејҸ: йҖҡиҝҮзӣ‘жҺ§вҪ№з»ңеҮәвјқзҡ„жөҒйҮҸпјҢеҲҶжһҗеҲӨж–ӯе“ӘдәӣеҶ…е®№жҳҜеҸҜд»ҘиҝӣвҫҸеҠ«жҢҒеӨ„зҗҶзҡ„,еҶҚеҜ№еҠ«жҢҒзҡ„еҶ…еӯҳеҸ‘иө·302и·іиҪ¬зҡ„еӣһеӨҚпјҢеј•еҜјвҪӨжҲ·иҺ·еҸ–еҶ…е®№

пјҲ2пјүHTTPеҠ«жҢҒ: (и®ҝй—®вҫ•жӯҢдҪҶжҳҜвјҖзӣҙжңүиҙӘзҺ©и“қвҪүзҡ„вјҙе‘Ҡ),з”ұдәҺhttpжҳҺвҪӮдј иҫ“,иҝҗиҗҘе•Ҷдјҡдҝ®ж”№дҪ зҡ„httpе“Қеә”еҶ…е®№(еҚіеҠ вјҙе‘Ҡ)
