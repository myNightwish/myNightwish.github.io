жӯӨеӨ–пјҢеҹәдәҺHookзҡ„и®Ёи®әеұ•ејҖд№ӢеүҚпјҢй“әеһ«дәҶдёҖзҜҮеҮҪж•°з»„д»¶VSзұ»з»„д»¶зҡ„зү№жҖ§еҲҶжһҗпјҢд»ҘеҸҠдёәд»Җд№ҲеҮҪж•°з»„д»¶и·ҹеҘ‘еҗҲreactзҡ„зҗҶеҝөпјҡ

  • зӣ®зҡ„пјҡеӯҰд№ HOOKзҡ„зү№жҖ§пјҢд»ҘеҸҠејҖеҸ‘е®һи·өж—¶жңүд»Җд№ҲеҗҜеҸ‘пјҲеҶҷжі•гҖҒдјҳеҢ–гҖҒжҺ’жҹҘbugзӯүпјү

whatВ·

  • жҳҜдёҖдәӣеҸҜд»Ҙи®©дҪ еңЁеҮҪж•°з»„д»¶йҮҢвҖңй’©е…ҘвҖқ React state еҸҠз”ҹе‘Ҫе‘Ёжңҹзӯүзү№жҖ§зҡ„еҮҪж•°

  • жҳҜдёҖеҘ—дҪҝеҮҪж•°з»„д»¶жӣҙејәеӨ§гҖҒжӣҙзҒөжҙ»зҡ„й’©еӯҗгҖӮеҮҪж•°з»„д»¶зӣёжҜ”дәҺзұ»з»„д»¶е°‘дәҶеҫҲеӨҡдёңиҘҝпјҢиҖҢHooksзҡ„еҮәзҺ°е°ұжҳҜеё®еҠ©еҮҪж•°ејҸз»„д»¶иЎҘйҪҗиҝҷз§Қзјәйҷ·гҖӮеҮҪж•°ејҸз»„д»¶еҸҜд»ҘиҮӘз”ұзҡ„дҪҝз”ЁHooksжҸҗдҫӣзҡ„дё°еҜҢе·Ҙе…·

1гҖҒuseStateеҹәзЎҖВ·

1.1 еҹәзЎҖз”Ёжі•В·

  • ж—©жңҹеҮҪж•°з»„д»¶зӣёжҜ”дәҺзұ»з»„д»¶пјҢеҠЈеҠҝд№ӢдёҖпјҡзјәд№Ҹз»ҙжҠӨе’Ңе®ҡд№үstateзҡ„иғҪеҠӣпјҢиҖҢиҝҷдёӘAPIдёәеҮҪж•°з»„д»¶еј•е…ҘзҠ¶жҖҒ
  • е®ғдёҺ class йҮҢйқўзҡ„ this.state жҸҗдҫӣзҡ„еҠҹиғҪе®Ңе…ЁзӣёеҗҢгҖӮдёҖиҲ¬жқҘиҜҙпјҢеңЁеҮҪж•°йҖҖеҮәеҗҺеҸҳйҮҸе°ұдјҡвҖқж¶ҲеӨұвҖқпјҢдҪҶ state дёӯзҡ„еҸҳйҮҸдјҡиў« React дҝқз•ҷгҖӮ
  • React дјҡеңЁйҮҚеӨҚжёІжҹ“ж—¶и®°дҪҸе®ғеҪ“еүҚзҡ„еҖјпјҢе№¶дё”жҸҗдҫӣжңҖж–°зҡ„еҖјз»ҷжҲ‘д»¬зҡ„еҮҪж•°гҖӮ

  • дёҺзұ»з»„д»¶зҡ„дёҚеҗҢпјҡ

    • еҗҢж ·йҖ»иҫ‘зҡ„еҮҪж•°з»„д»¶жҜ”зұ»з»„д»¶еӨҚжқӮеәҰжӣҙдҪҺпјҢд»Јз ҒйҮҸжӣҙе°‘
    • е®ғзұ»дјј class з»„д»¶зҡ„ this.setStateпјҢдҪҶе®ғдёҚдјҡжҠҠж–°state е’Ңж—§state еҗҲе№¶пјҢиҖҢжҳҜжӣҝжҚў
    • еҖјеҫ—жіЁж„Ҹзҡ„жҳҜпјҢдёҚеҗҢдәҺ this.stateпјҢиҝҷйҮҢзҡ„ state дёҚдёҖе®ҡиҰҒжҳҜдёҖдёӘеҜ№иұЎ вҖ”вҖ” еҰӮжһңдҪ жңүйңҖиҰҒпјҢе®ғд№ҹеҸҜд»ҘжҳҜгҖӮ
    • иҝҷдёӘеҲқе§Ӣ state еҸӮж•°еҸӘжңүеңЁз¬¬дёҖж¬ЎжёІжҹ“ж—¶дјҡиў«з”ЁеҲ°пјҢеңЁеҗҺз»ӯзҡ„йҮҚж–°жёІжҹ“дёӯпјҢuseState иҝ”еӣһзҡ„з¬¬дёҖдёӘеҖје°Ҷе§Ӣз»ҲжҳҜжӣҙж–°еҗҺжңҖж–°зҡ„ state

    жіЁж„Ҹ

    React дјҡзЎ®дҝқ setState еҮҪж•°зҡ„ж ҮиҜҶжҳҜзЁіе®ҡзҡ„пјҢе№¶дёҚдјҡеңЁз»„д»¶йҮҚж–°жёІжҹ“ж—¶еҸ‘з”ҹеҸҳеҢ–гҖӮиҝҷе°ұжҳҜдёәд»Җд№ҲеҸҜд»Ҙе®үе…Ёең°д»Һ useEffect жҲ– useCallback зҡ„дҫқиө–еҲ—иЎЁдёӯзңҒз•Ҙ setStateгҖӮ

1.2 еҮҪж•°ејҸsetStateжӣҙж–°В·

  • еҰӮжһңж–°state йңҖиҰҒйҖҡиҝҮз”Ёе…ҲеүҚstate и®Ўз®—еҫ—еҮәпјҢеҸҜд»Ҙе°ҶеҮҪж•°дј йҖ’з»ҷ setStateгҖӮ

  • иҜҘеҮҪж•°е°ҶжҺҘж”¶е…ҲеүҚstateпјҢе№¶иҝ”еӣһдёҖдёӘжӣҙж–°еҗҺзҡ„еҖјгҖӮ

  • еҰӮжһңдҪ зҡ„жӣҙж–°еҮҪж•°иҝ”еӣһеҖјдёҺеҪ“еүҚ state е®Ңе…ЁзӣёеҗҢпјҢеҲҷйҡҸеҗҺзҡ„йҮҚжёІжҹ“дјҡиў«е®Ңе…Ёи·іиҝҮгҖӮ

жіЁж„Ҹ

дёҺ class з»„д»¶дёӯзҡ„ setState ж–№жі•дёҚеҗҢпјҢuseState дёҚдјҡиҮӘеҠЁеҗҲе№¶жӣҙж–°еҜ№иұЎгҖӮдҪ еҸҜд»Ҙз”ЁеҮҪж•°ејҸзҡ„ setState з»“еҗҲеұ•ејҖиҝҗз®—з¬ҰжқҘиҫҫеҲ°еҗҲе№¶жӣҙж–°еҜ№иұЎзҡ„ж•ҲжһңгҖӮ

useReducer жҳҜеҸҰдёҖз§ҚеҸҜйҖүж–№жЎҲпјҢе®ғжӣҙйҖӮеҗҲз”ЁдәҺз®ЎзҗҶеҢ…еҗ«еӨҡдёӘеӯҗеҖјзҡ„ state еҜ№иұЎгҖӮ

1.3 жғ°жҖ§initialStateВ·

  • initialState еҸӮж•°еҸӘдјҡеңЁз»„д»¶зҡ„еҲқе§ӢжёІжҹ“дёӯиө·дҪңз”ЁпјҢеҗҺз»ӯжёІжҹ“ж—¶дјҡиў«еҝҪз•ҘгҖӮ

  • еҰӮжһңеҲқе§Ӣ state йңҖйҖҡиҝҮеӨҚжқӮи®Ўз®—иҺ·еҫ—пјҢеҲҷеҸҜдј е…ҘдёҖдёӘеҮҪж•°пјҢеңЁеҮҪж•°дёӯи®Ўз®—е№¶иҝ”еӣһеҲқе§Ӣзҡ„ stateпјҢжӯӨеҮҪж•°еҸӘеңЁеҲқе§ӢжёІжҹ“ж—¶иў«и°ғз”Ёпјҡ

1.4 и·іиҝҮ state жӣҙж–°В·

  • и°ғз”Ё State Hook зҡ„жӣҙж–°еҮҪж•°е№¶дј е…ҘеҪ“еүҚзҡ„ state ж—¶пјҢд№ҹе°ұжҳҜеҮҪж•°ејҸжӣҙж–°пјҢstateиҝ”еӣһеҖјжІЎжңүеҸ‘з”ҹеҸҳеҢ–ж—¶пјҢReact е°Ҷи·іиҝҮеӯҗз»„д»¶зҡ„жёІжҹ“еҸҠ effect зҡ„жү§иЎҢгҖӮпјҲReact дҪҝз”Ё Object.is жҜ”иҫғз®—жі• жқҘжҜ”иҫғ stateгҖӮпјү

йңҖиҰҒжіЁж„Ҹзҡ„жҳҜпјҡ

  • React еҸҜиғҪд»ҚйңҖеңЁи·іиҝҮжёІжҹ“еүҚжёІжҹ“иҜҘз»„д»¶гҖӮдёҚиҝҮз”ұдәҺ React дёҚдјҡеҜ№з»„д»¶ж ‘зҡ„вҖңж·ұеұӮвҖқиҠӮзӮ№иҝӣиЎҢдёҚеҝ…иҰҒзҡ„жёІжҹ“пјҢжүҖд»ҘеӨ§еҸҜдёҚеҝ…жӢ…еҝғгҖӮ

  • еҰӮжһңдҪ еңЁжёІжҹ“жңҹй—ҙжү§иЎҢдәҶй«ҳејҖй”Җзҡ„и®Ўз®—пјҢеҲҷеҸҜз”Ё useMemo жқҘиҝӣиЎҢдјҳеҢ–гҖӮ

1.5 жіЁж„ҸпјҡВ·

  • еӨҡдёӘзҠ¶жҖҒеЈ°жҳҺж—¶пјҢеҝ…йЎ»еҲҶејҖеҶҷ

  • еӨҡж¬Ўи°ғз”Ё useState ж—¶пјҢеҝ…йЎ»дҝқиҜҒжҜҸж¬ЎжёІжҹ“ж—¶е®ғд»¬зҡ„и°ғз”ЁйЎәеәҸжҳҜдёҚеҸҳзҡ„гҖӮеҗҺйқўи§ЈйҮҠwhy

  • еӣ жӯӨпјҢдёҚиғҪеҮәзҺ°еңЁжқЎд»¶еҲӨж–ӯиҜӯеҸҘдёӯпјҢеӣ дёәе®ғеҝ…йЎ»жңүдёҖж ·зҡ„жёІжҹ“йЎәеәҸ

  • why йЎәеәҸдёәд»Җд№ҲйҮҚиҰҒпјҹВ·

    е·Із»ҸеЎ«еқ‘вҖ”вҖ”иҜ·жҹҘзңӢHookзҡ„дҪҝз”Ёи§„еҲҷе°ҸиҠӮйғЁеҲҶжҲ–иҖ…жәҗз Ғи°ғз”ЁжөҒзЁӢйғЁеҲҶпјӣ

2гҖҒ useEffectеҹәВ·

д№ӢеүҚеңЁReact з»„д»¶дёӯжү§иЎҢиҝҮж•°жҚ®иҺ·еҸ–гҖҒи®ўйҳ…жҲ–иҖ…жүӢеҠЁдҝ®ж”№иҝҮ DOMгҖӮжҲ‘д»¬з»ҹдёҖжҠҠиҝҷдәӣж“ҚдҪңз§°дёә**вҖңеүҜдҪңз”ЁвҖқ**

  • useEffect е°ұжҳҜдёҖдёӘ Effect HookпјҢз»ҷеҮҪж•°з»„д»¶еўһеҠ дәҶж“ҚдҪңеүҜдҪңз”Ёзҡ„иғҪеҠӣ
  • е®ғи·ҹ class з»„д»¶дёӯзҡ„ componentDidMountгҖҒcomponentDidUpdate е’Ң componentWillUnmount е…·жңүзӣёеҗҢзҡ„з”ЁйҖ”пјҢеҸӘдёҚиҝҮиў«еҗҲе№¶жҲҗдәҶдёҖдёӘ API
  • и°ғз”Ё useEffect ж—¶пјҢе°ұжҳҜеңЁе‘ҠиҜү React еңЁе®ҢжҲҗеҜ№ DOM зҡ„жӣҙж”№еҗҺиҝҗиЎҢдҪ зҡ„вҖңеүҜдҪңз”ЁвҖқеҮҪж•°гҖӮз”ұдәҺеүҜдҪңз”ЁеҮҪж•°жҳҜеңЁз»„д»¶еҶ…еЈ°жҳҺзҡ„пјҢжүҖд»ҘеҸҜд»Ҙи®ҝй—®еҲ°з»„д»¶зҡ„ props е’Ң state

2.1 еј•е…ҘеүҜдҪңз”ЁВ·

з”ҹе‘Ҫе‘ЁжңҹеҲ°useEffectзҡ„иҪ¬жҚўе…ізі»йғҪдёҚжҳҜжңҖйҮҚиҰҒзҡ„пјҢжңҖйҮҚиҰҒзҡ„жҳҜжһ„е»әз»„д»¶жңүеүҜдҪңз”ЁпјҢuseEffectдёәеҮҪж•°з»„д»¶еј•е…ҘеүҜдҪңз”Ёзҡ„й’©еӯҗ

  • йҖҡиҝҮ HookпјҢеҸҜд»ҘжҠҠз»„д»¶еҶ…зӣёе…ізҡ„еүҜдҪңз”Ёз»„з»ҮеңЁдёҖиө·пјҲдҫӢеҰӮеҲӣе»әи®ўйҳ…еҸҠеҸ–ж¶Ҳи®ўйҳ…пјүпјҢиҖҢдёҚиҰҒжҠҠе®ғд»¬жӢҶеҲҶеҲ°дёҚеҗҢзҡ„з”ҹе‘Ҫе‘ЁжңҹеҮҪж•°йҮҢ

  • иҝҮеҺ»еңЁDidMount гҖҒDidUpdateгҖҒwillUnmountдёӯеҒҡзҡ„дәӢпјҢеҸҜж”ҫеңЁиҜҘй’©еӯҗеҒҡгҖӮжҜ”еҰӮпјҡж“ҚдҪңDOMгҖҒиҺ·еҸ–еӨ–йғЁAPI

  • гҖҗжіЁж„ҸгҖ‘пјҡuseEffectдёӯиҝ”еӣһзҡ„еҮҪж•°еҸ«вҖңжё…йҷӨеҮҪж•°вҖқпјҡ

    еҪ“ReactиҜҶеҲ«еҲ°иҜҘеҮҪж•°пјҢдјҡеңЁеҚёиҪҪйҳ¶ж®өжү§иЎҢе…¶жё…йҷӨйҖ»иҫ‘гҖӮ

    иҝҷдёӘеҮҪж•°дёҚдјҡж”¶з¬¬2дёӘеҸӮж•°гҖҒжҲ–е…¶д»–еҸӮж•°еӣ зҙ зҡ„еҪұе“ҚгҖӮ

    еҸӘиҰҒдҪ еңЁuseEffectеӣһи°ғдёӯиҝ”еӣһдәҶдёҖдёӘеҮҪж•°пјҢе®ғе°ұдјҡиў«еҪ“еҒҡжё…йҷӨеҮҪж•°жқҘеӨ„зҗҶгҖӮ

2.2 ж— йңҖжё…йҷӨзҡ„ effect вӯҗпёҸйқўиҜ•~В·

жңүж—¶еҖҷпјҢжҲ‘д»¬еҸӘжғі**еңЁ React жӣҙж–° DOM д№ӢеҗҺиҝҗиЎҢдёҖдәӣйўқеӨ–зҡ„д»Јз ҒгҖӮ**жҜ”еҰӮеҸ‘йҖҒзҪ‘з»ңиҜ·жұӮпјҢжүӢеҠЁеҸҳжӣҙ DOMпјҢи®°еҪ•ж—Ҙеҝ—пјҢиҝҷдәӣйғҪжҳҜеёёи§Ғзҡ„ж— йңҖжё…йҷӨзҡ„ж“ҚдҪңгҖӮ

еӣ дёәеңЁжү§иЎҢе®Ңиҝҷдәӣж“ҚдҪңд№ӢеҗҺпјҢе°ұеҸҜд»ҘеҝҪз•Ҙд»–д»¬дәҶгҖӮ

  • useEffect еҒҡдәҶд»Җд№Ҳпјҹ React дјҡдҝқеӯҳдҪ дј йҖ’зҡ„еҮҪж•°пјҲeffectпјүпјҢе№¶еңЁжү§иЎҢ DOM жӣҙж–°д№ӢеҗҺи°ғз”Ёе®ғ

  • дёәд»Җд№ҲеңЁз»„д»¶еҶ…йғЁи°ғз”Ё useEffectпјҹ

    еҸҜд»ҘеңЁ effect дёӯзӣҙжҺҘи®ҝй—® count state еҸҳйҮҸпјҲжҲ–е…¶д»– propsпјүпјҢиҖҢдёҚйңҖиҰҒзү№ж®Ҡзҡ„ API жқҘиҜ»еҸ–е®ғпјҢе®ғе·Із»ҸдҝқеӯҳеңЁеҮҪж•°дҪңз”ЁеҹҹдёӯпјҲHook дҪҝз”ЁдәҶJSзҡ„й—ӯеҢ…жңәеҲ¶пјү

  • useEffect дјҡеңЁжҜҸж¬ЎжёІжҹ“еҗҺйғҪжү§иЎҢеҗ—пјҹ

    жҳҜзҡ„пјҢй»ҳи®Өжғ…еҶөдёӢпјҢе®ғеңЁз¬¬дёҖж¬ЎжёІжҹ“д№ӢеҗҺе’ҢжҜҸж¬Ўжӣҙж–°д№ӢеҗҺйғҪдјҡжү§иЎҢгҖӮ

    дҪ еҸҜиғҪдјҡжӣҙе®№жҳ“жҺҘеҸ— effect еҸ‘з”ҹеңЁвҖңжёІжҹ“д№ӢеҗҺвҖқиҝҷз§ҚжҰӮеҝөпјҢдёҚз”ЁеҶҚеҺ»иҖғиҷ‘вҖңжҢӮиҪҪвҖқиҝҳжҳҜвҖңжӣҙж–°вҖқгҖӮ

    React дҝқиҜҒдәҶжҜҸж¬ЎиҝҗиЎҢ effect зҡ„еҗҢж—¶пјҢDOM йғҪе·Із»Ҹжӣҙж–°е®ҢжҜ•гҖӮ

  • еҮҪж•°еңЁжёІжҹ“иҝҮзЁӢдёӯзҡ„дёҚеҗҢпјҡвӯҗпёҸвӯҗпёҸвӯҗпёҸВ·
    • жіЁж„ҸеҲ°пјҢдј йҖ’з»ҷ useEffect зҡ„еҮҪж•°еңЁжҜҸж¬ЎжёІжҹ“дёӯйғҪдјҡжңүжүҖдёҚеҗҢпјҢиҝҷжҳҜеҲ»ж„Ҹдёәд№Ӣзҡ„гҖӮ
    • дәӢе®һдёҠиҝҷжӯЈжҳҜжҲ‘д»¬еҸҜд»ҘеңЁ effect дёӯиҺ·еҸ–жңҖж–°зҡ„ count зҡ„еҖјпјҢиҖҢдёҚз”ЁжӢ…еҝғе…¶иҝҮжңҹзҡ„еҺҹеӣ гҖӮ
    • жҜҸж¬ЎжҲ‘д»¬йҮҚж–°жёІжҹ“пјҢйғҪдјҡз”ҹжҲҗж–°зҡ„ effectпјҢжӣҝжҚўжҺүд№ӢеүҚзҡ„гҖӮжҹҗз§Қж„Ҹд№үдёҠи®ІпјҢeffect жӣҙеғҸжҳҜжёІжҹ“з»“жһңзҡ„дёҖйғЁеҲҶ вҖ”вҖ” жҜҸдёӘ effect вҖңеұһдәҺвҖқдёҖж¬Ўзү№е®ҡзҡ„жёІжҹ“гҖӮ
    • дёҺ componentDidMount жҲ– componentDidUpdate дёҚеҗҢпјҢдҪҝз”Ё useEffect и°ғеәҰзҡ„ effect дёҚдјҡйҳ»еЎһжөҸи§ҲеҷЁжӣҙж–°еұҸе№•пјҢиҝҷи®©дҪ зҡ„еә”з”ЁзңӢиө·жқҘе“Қеә”жӣҙеҝ«гҖӮеӨ§еӨҡж•°жғ…еҶөдёӢпјҢeffect дёҚйңҖиҰҒеҗҢжӯҘең°жү§иЎҢгҖӮеңЁдёӘеҲ«жғ…еҶөдёӢпјҲдҫӢеҰӮжөӢйҮҸеёғеұҖпјүпјҢжңүеҚ•зӢ¬зҡ„ useLayoutEffect Hook дҫӣдҪ дҪҝз”ЁпјҢе…¶ API дёҺ useEffect зӣёеҗҢгҖӮ
    • еҗҺз»ӯз« иҠӮжҢ–еқ‘пјҡвӯҗпёҸ

2.3 йңҖжё…йҷӨзҡ„effectВ·

жӯӨеӨ–пјҢиҝҳжңүдёҖдәӣеүҜдҪңз”ЁжҳҜйңҖиҰҒжё…йҷӨзҡ„гҖӮдҫӢеҰӮи®ўйҳ…еӨ–йғЁж•°жҚ®жәҗгҖӮиҝҷз§Қжғ…еҶөдёӢпјҢжё…йҷӨе·ҘдҪңеҸҜд»ҘйҳІжӯўеј•иө·еҶ…еӯҳжі„йңІпјҒ

  • дёәд»Җд№ҲиҰҒеңЁ effect дёӯиҝ”еӣһдёҖдёӘеҮҪж•°пјҹ

    иҝҷжҳҜ effect еҸҜйҖүзҡ„жё…йҷӨжңәеҲ¶гҖӮжҜҸдёӘ effect йғҪеҸҜд»Ҙиҝ”еӣһдёҖдёӘжё…йҷӨеҮҪж•°гҖӮеҰӮжӯӨеҸҜд»Ҙе°Ҷж·»еҠ е’Ңз§»йҷӨи®ўйҳ…зҡ„йҖ»иҫ‘ж”ҫеңЁдёҖиө·гҖӮе®ғд»¬йғҪеұһдәҺ effect зҡ„дёҖйғЁеҲҶгҖӮ

  • React дҪ•ж—¶жё…йҷӨ effectпјҹ

    • дёәйҳІжӯўеҶ…еӯҳжі„жјҸпјҢжё…йҷӨеҮҪж•°дјҡеңЁз»„д»¶еҚёиҪҪеүҚжү§иЎҢгҖӮеҰӮжһңз»„д»¶еӨҡж¬ЎжёІжҹ“пјҲйҖҡеёёеҰӮжӯӨпјүпјҢеҲҷеңЁжү§иЎҢдёӢдёҖдёӘ effect д№ӢеүҚпјҢдёҠдёҖдёӘ effect е°ұе·Іиў«жё…йҷӨгҖӮеңЁдёҠиҝ°зӨәдҫӢдёӯпјҢж„Ҹе‘ізқҖз»„д»¶зҡ„жҜҸдёҖж¬Ўжӣҙж–°йғҪдјҡеҲӣе»әж–°зҡ„и®ўйҳ…гҖӮиӢҘжғійҒҝе…ҚжҜҸж¬Ўжӣҙж–°йғҪи§ҰеҸ‘ effect зҡ„жү§иЎҢ

    • жӯЈеҰӮд№ӢеүҚеӯҰеҲ°зҡ„пјҢeffect еңЁжҜҸж¬ЎжёІжҹ“ж—¶йғҪдјҡжү§иЎҢгҖӮиҝҷе°ұжҳҜдёәд»Җд№Ҳ React дјҡеңЁжү§иЎҢеҪ“еүҚ effect д№ӢеүҚжё…йҷӨдёҠдёҖдёӘ effect

    дёәд»Җд№ҲиҰҒиҝҷж ·пјҢд»ҘеҸҠеҰӮдҪ•еңЁйҒҮеҲ°жҖ§иғҪй—®йўҳж—¶и·іиҝҮжӯӨиЎҢдёәеңЁ2.5 2.6дјҡеұ•ејҖ

2.4 Effectзҡ„жү§иЎҢж—¶жңәВ·

  • дёҺ componentDidMountгҖҒcomponentDidUpdate дёҚеҗҢзҡ„жҳҜпјҢдј з»ҷ useEffect зҡ„еҮҪж•°дјҡеңЁжөҸи§ҲеҷЁе®ҢжҲҗеёғеұҖдёҺз»ҳеҲ¶д№ӢеҗҺпјҢеңЁдёҖдёӘе»¶иҝҹдәӢд»¶дёӯиў«и°ғз”ЁгҖӮ

  • иҝҷдҪҝеҫ—е®ғйҖӮз”ЁдәҺи®ёеӨҡеёёи§Ғзҡ„еүҜдҪңз”ЁеңәжҷҜпјҢжҜ”еҰӮи®ҫзҪ®и®ўйҳ…е’ҢдәӢд»¶еӨ„зҗҶзӯүжғ…еҶөпјҢеӣ дёәз»қеӨ§еӨҡж•°ж“ҚдҪңдёҚеә”йҳ»еЎһжөҸи§ҲеҷЁеҜ№еұҸе№•зҡ„жӣҙж–°гҖӮ

  • з„¶иҖҢпјҢе№¶йқһжүҖжңү effect йғҪеҸҜд»Ҙиў«е»¶иҝҹжү§иЎҢгҖӮдҫӢеҰӮпјҢдёҖдёӘеҜ№з”ЁжҲ·еҸҜи§Ғзҡ„ DOM еҸҳжӣҙе°ұеҝ…йЎ»еңЁжөҸи§ҲеҷЁжү§иЎҢдёӢдёҖж¬Ўз»ҳеҲ¶еүҚиў«еҗҢжӯҘжү§иЎҢпјҢиҝҷж ·з”ЁжҲ·жүҚдёҚдјҡж„ҹи§үеҲ°и§Ҷи§үдёҠзҡ„дёҚдёҖиҮҙгҖӮ

  • React дёәжӯӨжҸҗдҫӣдәҶдёҖдёӘйўқеӨ–зҡ„ useLayoutEffect Hook жқҘеӨ„зҗҶиҝҷзұ» effectгҖӮе®ғе’Ң useEffect зҡ„з»“жһ„зӣёеҗҢпјҢеҢәеҲ«еҸӘжҳҜи°ғз”Ёж—¶жңәдёҚеҗҢгҖӮ

  • иҷҪз„¶ useEffect дјҡеңЁжөҸи§ҲеҷЁз»ҳеҲ¶еҗҺе»¶иҝҹжү§иЎҢпјҢдҪҶдјҡдҝқиҜҒеңЁд»»дҪ•ж–°зҡ„жёІжҹ“еүҚжү§иЎҢгҖӮеңЁејҖе§Ӣж–°зҡ„жӣҙж–°еүҚпјҢReact жҖ»дјҡе…Ҳжё…йҷӨдёҠдёҖиҪ®жёІжҹ“зҡ„ effectгҖӮ

дҪҝз”ЁеӨҡдёӘEffectе®һзҺ°е…іжіЁзӮ№еҲҶзҰ»В·

  • дҪҝз”Ё Hook е…¶дёӯдёҖдёӘзӣ®зҡ„е°ұжҳҜиҰҒи§ЈеҶі class дёӯз”ҹе‘Ҫе‘ЁжңҹеҮҪж•°з»ҸеёёеҢ…еҗ«дёҚзӣёе…ізҡ„йҖ»иҫ‘пјҢдҪҶеҸҲжҠҠзӣёе…ійҖ»иҫ‘еҲҶзҰ»еҲ°дәҶеҮ дёӘдёҚеҗҢж–№жі•дёӯзҡ„й—®йўҳгҖӮsoпјҢе®ғеҸҜд»ҘдҪҝз”ЁеӨҡдёӘ effectгҖӮиҝҷдјҡе°ҶдёҚзӣёе…ійҖ»иҫ‘еҲҶзҰ»еҲ°дёҚеҗҢзҡ„ effect дёӯпјҡ
  • Hook е…Ғи®ёжҲ‘д»¬жҢүз…§д»Јз Ғзҡ„з”ЁйҖ”еҲҶзҰ»д»–д»¬пјҢ иҖҢдёҚжҳҜеғҸз”ҹе‘Ҫе‘ЁжңҹеҮҪж•°йӮЈж ·гҖӮReact е°ҶжҢүз…§ effect еЈ°жҳҺзҡ„йЎәеәҸдҫқж¬Ўи°ғз”Ёз»„д»¶дёӯзҡ„жҜҸдёҖдёӘ effectгҖӮ

2.5 дёәдҪ•жҜҸж¬Ўжӣҙж–°ж—¶йғҪиҝҗиЎҢ EffectВ·

дёәд»Җд№Ҳ effect зҡ„жё…йҷӨйҳ¶ж®өеңЁжҜҸж¬ЎйҮҚж–°жёІжҹ“ж—¶йғҪдјҡжү§иЎҢпјҢиҖҢдёҚжҳҜеҸӘеңЁеҚёиҪҪз»„д»¶зҡ„ж—¶еҖҷжү§иЎҢдёҖж¬ЎгҖӮ

  • еӣ дёәиҝҷж ·зҡ„и®ҫи®ЎеҸҜд»Ҙеё®еҠ©еҲӣе»әз»„д»¶bugжӣҙе°‘пјҡ

    дёҖдёӘз”ЁдәҺжҳҫзӨәеҘҪеҸӢжҳҜеҗҰеңЁзәҝзҡ„ FriendStatus з»„д»¶гҖӮд»Һ class дёӯ props иҜ»еҸ– friend.idпјҢз„¶еҗҺеңЁз»„д»¶жҢӮиҪҪеҗҺи®ўйҳ…еҘҪеҸӢзҡ„зҠ¶жҖҒпјҢе№¶еңЁеҚёиҪҪз»„д»¶зҡ„ж—¶еҖҷеҸ–ж¶Ҳи®ўйҳ…пјҡ

    дҪҶеҪ“з»„д»¶е·ІжҳҫзӨәеңЁеұҸе№•ж—¶пјҢfriend prop еҸ‘з”ҹеҸҳеҢ–ж—¶дјҡеҸ‘з”ҹд»Җд№Ҳпјҹ

    жҲ‘д»¬зҡ„з»„д»¶е°Ҷз»§з»ӯеұ•зӨәеҺҹжқҘзҡ„еҘҪеҸӢзҠ¶жҖҒгҖӮиҝҷжҳҜдёҖдёӘ bugгҖӮ

    иҖҢдё”жҲ‘д»¬иҝҳдјҡеӣ дёәеҸ–ж¶Ҳи®ўйҳ…ж—¶дҪҝз”Ёй”ҷиҜҜзҡ„еҘҪеҸӢ ID еҜјиҮҙеҶ…еӯҳжі„йңІжҲ–еҙ©жәғзҡ„й—®йўҳгҖӮclass з»„д»¶дёӯиҰҒж·»еҠ  componentDidUpdate жқҘи§ЈеҶіиҝҷдёӘй—®йўҳпјҡ

    еҝҳи®°жӯЈзЎ®ең°еӨ„зҗҶ componentDidUpdate жҳҜ React еә”з”Ёдёӯеёёи§Ғзҡ„ bug жқҘжәҗпјҢе…¶е®һзЎ®е®һеӯҳеңЁ

  • дҪҶhookзҡ„еҶҷжі•пјҡе№¶дёҚдјҡеҸ—еҲ°жӯӨ bug еҪұе“Қ

    е№¶дёҚйңҖзү№е®ҡзҡ„д»Јз ҒжқҘеӨ„зҗҶжӣҙж–°йҖ»иҫ‘пјҢеӣ дёә useEffect й»ҳи®Өе°ұдјҡеӨ„зҗҶгҖӮе®ғдјҡеңЁи°ғз”ЁдёҖдёӘж–°зҡ„ effect д№ӢеүҚеҜ№еүҚдёҖдёӘ effect иҝӣиЎҢжё…зҗҶгҖӮдёӢйқўжҢүж—¶й—ҙеҲ—еҮәдёҖдёӘеҸҜиғҪдјҡдә§з”ҹзҡ„и®ўйҳ…е’ҢеҸ–ж¶Ҳи®ўйҳ…ж“ҚдҪңи°ғз”ЁеәҸеҲ—жқҘиҜҙжҳҺпјҡ

    жӯӨй»ҳи®ӨиЎҢдёәдҝқиҜҒдәҶдёҖиҮҙжҖ§пјҢйҒҝе…ҚдәҶеңЁ class з»„д»¶дёӯеӣ дёәжІЎжңүеӨ„зҗҶжӣҙж–°йҖ»иҫ‘иҖҢеҜјиҮҙеёёи§Ғзҡ„ bugгҖӮ

2.6 йҖҡиҝҮи·іиҝҮ EffectжқҘжҖ§иғҪдјҳеҢ–В·

еңЁжҹҗдәӣжғ…еҶөдёӢпјҢжҜҸж¬ЎжёІжҹ“еҗҺйғҪжү§иЎҢжё…зҗҶжҲ–жү§иЎҢ effect еҸҜиғҪдјҡеҜјиҮҙжҖ§иғҪй—®йўҳгҖӮ

жёІжҹ“пјҡВ·

  • еңЁ class з»„д»¶дёӯпјҢйҖҡиҝҮеңЁ componentDidUpdate дёӯж·»еҠ еҜ№ prevProps жҲ– prevState зҡ„жҜ”иҫғйҖ»иҫ‘и§ЈеҶіпјҡ

  • иҝҷжҳҜеҫҲеёёи§Ғзҡ„йңҖжұӮпјҢжүҖд»Ҙе®ғиў«еҶ…зҪ®еҲ°дәҶ useEffect зҡ„ Hook API дёӯгҖӮеҰӮжһңжҹҗдәӣзү№е®ҡеҖјеңЁдёӨж¬ЎйҮҚжёІжҹ“д№Ӣй—ҙжІЎжңүеҸ‘з”ҹеҸҳеҢ–пјҢдҪ еҸҜд»ҘйҖҡзҹҘ React и·іиҝҮеҜ№ effect зҡ„и°ғз”ЁпјҢеҸӘиҰҒдј йҖ’ж•°з»„дҪңдёә useEffect зҡ„з¬¬дәҢдёӘеҸҜйҖүеҸӮж•°еҚіеҸҜпјҡ

    еҰӮжһңж•°з»„дёӯжңүеӨҡдёӘе…ғзҙ пјҢеҚідҪҝеҸӘжңүдёҖдёӘе…ғзҙ еҸ‘з”ҹеҸҳеҢ–пјҢReact д№ҹдјҡжү§иЎҢ effectгҖӮ

жё…йҷӨж—¶пјҡВ·

  • еҜ№дәҺжңүжё…йҷӨж“ҚдҪңзҡ„ effect еҗҢж ·йҖӮз”ЁпјҡжңӘжқҘзүҲжң¬пјҢеҸҜиғҪдјҡеңЁжһ„е»әж—¶иҮӘеҠЁж·»еҠ з¬¬дәҢдёӘеҸӮж•°гҖӮ

  • еҰӮжһңдҪ иҰҒдҪҝз”ЁжӯӨдјҳеҢ–ж–№ејҸпјҢиҜ·зЎ®дҝқж•°з»„дёӯеҢ…еҗ«дәҶжүҖжңүеӨ–йғЁдҪңз”ЁеҹҹдёӯдјҡйҡҸж—¶й—ҙеҸҳеҢ–е№¶дё”еңЁ effect дёӯдҪҝз”Ёзҡ„еҸҳйҮҸпјҢеҗҰеҲҷдҪ зҡ„д»Јз Ғдјҡеј•з”ЁеҲ°е…ҲеүҚжёІжҹ“дёӯзҡ„ж—§еҸҳйҮҸгҖӮеҸӮйҳ…ж–ҮжЎЈпјҢдәҶи§ЈжӣҙеӨҡе…ідәҺеҰӮдҪ•еӨ„зҗҶеҮҪж•°д»ҘеҸҠж•°з»„йў‘з№ҒеҸҳеҢ–ж—¶зҡ„жҺӘж–ҪеҶ…е®№гҖӮ

  • еҰӮжһңжғіжү§иЎҢеҸӘиҝҗиЎҢдёҖж¬Ўзҡ„ effectпјҲд»…еңЁз»„д»¶жҢӮиҪҪе’ҢеҚёиҪҪж—¶жү§иЎҢпјүпјҢеҸҜд»Ҙдј йҖ’дёҖдёӘз©әж•°з»„пјҲ[]пјүдҪңдёәз¬¬дәҢдёӘеҸӮж•°гҖӮ

    • иҝҷе°ұе‘ҠиҜү React дҪ зҡ„ effect дёҚдҫқиө–дәҺ props жҲ– state дёӯзҡ„д»»дҪ•еҖјпјҢжүҖд»Ҙе®ғж°ёиҝңйғҪдёҚйңҖиҰҒйҮҚеӨҚжү§иЎҢгҖӮиҝҷе№¶дёҚеұһдәҺзү№ж®Ҡжғ…еҶө вҖ”вҖ” е®ғдҫқз„¶йҒөеҫӘдҫқиө–ж•°з»„зҡ„е·ҘдҪңж–№ејҸгҖӮ

    • йҷӨжӯӨд№ӢеӨ–пјҢиҜ·и®°еҫ— React дјҡзӯүеҫ…жөҸи§ҲеҷЁе®ҢжҲҗз”»йқўжёІжҹ“д№ӢеҗҺжүҚдјҡе»¶иҝҹи°ғз”Ё useEffectпјҢеӣ жӯӨдјҡдҪҝеҫ—йўқеӨ–ж“ҚдҪңеҫҲж–№дҫҝгҖӮ

    • жҺЁиҚҗеҗҜз”Ё eslint-plugin-react-hooks дёӯзҡ„ exhaustive-deps и§„еҲҷгҖӮжӯӨи§„еҲҷдјҡеңЁж·»еҠ й”ҷиҜҜдҫқиө–ж—¶еҸ‘еҮәиӯҰе‘Ҡе№¶з»ҷеҮәдҝ®еӨҚе»әи®®гҖӮ

4.1В·

еңЁжҹҗдәӣеңәжҷҜдёӢпјҢuseReducer дјҡжҜ” useState жӣҙйҖӮз”ЁпјҢдҫӢеҰӮ state йҖ»иҫ‘иҫғеӨҚжқӮдё”еҢ…еҗ«еӨҡдёӘеӯҗеҖјпјҢжҲ–иҖ…дёӢдёҖдёӘ state дҫқиө–дәҺд№ӢеүҚзҡ„ state зӯүгҖӮе№¶дё”пјҢдҪҝз”Ё useReducer иҝҳиғҪз»ҷйӮЈдәӣдјҡи§ҰеҸ‘ж·ұжӣҙж–°зҡ„з»„д»¶еҒҡжҖ§иғҪдјҳеҢ–пјҢеӣ дёәдҪ еҸҜд»Ҙеҗ‘еӯҗз»„д»¶дј йҖ’ dispatch иҖҢдёҚжҳҜеӣһи°ғеҮҪж•° гҖӮ

  • useState зҡ„жӣҝд»Јж–№жЎҲгҖӮе®ғжҺҘж”¶дёҖдёӘеҪўеҰӮ (state, action) => newState зҡ„ reducerпјҢе№¶иҝ”еӣһеҪ“еүҚзҡ„ state д»ҘеҸҠдёҺе…¶й…ҚеҘ—зҡ„ dispatch ж–№жі•гҖӮ

  • жіЁж„Ҹ

    React дјҡзЎ®дҝқ dispatch еҮҪж•°зҡ„ж ҮиҜҶжҳҜзЁіе®ҡзҡ„пјҢе№¶дёҚдјҡеңЁз»„д»¶йҮҚж–°жёІжҹ“ж—¶ж”№еҸҳгҖӮиҝҷе°ұжҳҜдёәд»Җд№ҲеҸҜд»Ҙе®үе…Ёең°д»Һ useEffect жҲ– useCallback зҡ„дҫқиө–еҲ—иЎЁдёӯзңҒз•Ҙ dispatchгҖӮ

4.3 жҢҮе®ҡеҲқе§Ӣ stateВ·

жңүдёӨз§ҚдёҚеҗҢеҲқе§ӢеҢ– useReducer state зҡ„ж–№ејҸпјҡ

    • React дёҚдҪҝз”Ё state = initialState иҝҷдёҖз”ұ Redux жҺЁе№ҝејҖжқҘзҡ„еҸӮж•°зәҰе®ҡгҖӮжңүж—¶еҖҷеҲқе§ӢеҖјдҫқиө–дәҺ propsпјҢеӣ жӯӨйңҖиҰҒеңЁи°ғз”Ё Hook ж—¶жҢҮе®ҡгҖӮ
    • еҰӮжһңдҪ зү№еҲ«е–ңж¬ўдёҠиҝ°зҡ„еҸӮж•°зәҰе®ҡпјҢеҸҜд»ҘйҖҡиҝҮи°ғз”Ё useReducer(reducer, undefined, reducer) жқҘжЁЎжӢҹ Redux зҡ„иЎҢдёәпјҢдҪҶдёҚйј“еҠұдҪ иҝҷд№ҲеҒҡгҖӮ

  • иҝҷд№ҲеҒҡеҸҜе°Ҷз”ЁдәҺи®Ўз®— state зҡ„йҖ»иҫ‘жҸҗеҲ°reducerеӨ–йғЁпјҢиҝҷд№ҹдёәе°ҶжқҘеҜ№йҮҚзҪ® state зҡ„ action еҒҡеӨ„зҗҶжҸҗдҫӣдәҶдҫҝеҲ©пјҡ

  • йңҖиҰҒжіЁж„Ҹзҡ„жҳҜпјҢReact еҸҜиғҪд»ҚйңҖиҰҒеңЁи·іиҝҮжёІжҹ“еүҚеҶҚж¬ЎжёІжҹ“иҜҘз»„д»¶гҖӮдёҚиҝҮз”ұдәҺ React дёҚдјҡеҜ№з»„д»¶ж ‘зҡ„вҖңж·ұеұӮвҖқиҠӮзӮ№иҝӣиЎҢдёҚеҝ…иҰҒзҡ„жёІжҹ“пјҢжүҖд»ҘеӨ§еҸҜдёҚеҝ…жӢ…еҝғгҖӮеҰӮжһңдҪ еңЁжёІжҹ“жңҹй—ҙжү§иЎҢдәҶй«ҳејҖй”Җзҡ„и®Ўз®—пјҢеҲҷеҸҜд»ҘдҪҝз”Ё useMemo жқҘиҝӣиЎҢдјҳеҢ–гҖӮдёҚжҳҜеҫҲзҗҶи§Ј~пјҹпјҹпјҹ

5гҖҒuseRefВ·

зү№жҖ§В·

  • useRef иҝ”еӣһдёҖдёӘеҸҜеҸҳзҡ„ ref еҜ№иұЎпјҢе…¶ .current еұһжҖ§иў«еҲқе§ӢеҢ–дёәдј е…Ҙзҡ„еҸӮж•°пјҲinitialValueпјүгҖӮиҝ”еӣһзҡ„ ref еҜ№иұЎеңЁз»„д»¶зҡ„ж•ҙдёӘз”ҹе‘Ҫе‘ЁжңҹеҶ…жҢҒз»ӯеӯҳеңЁгҖӮ
  • жң¬иҙЁдёҠпјҢuseRef е°ұеғҸжҳҜеҸҜд»ҘеңЁе…¶ .current еұһжҖ§дёӯдҝқеӯҳдёҖдёӘеҸҜеҸҳеҖјзҡ„вҖңзӣ’еӯҗвҖқгҖӮ
  • useRef() жҜ” ref еұһжҖ§жӣҙжңүз”Ёпјҡ
    • refпјҡжҳҜдёҖз§Қи®ҝй—® DOM зҡ„дё»иҰҒж–№ејҸпјҢеҰӮжһңе°Ҷ ref еҜ№иұЎд»Ҙ <div ref={myRef} /> еҪўејҸдј е…Ҙз»„д»¶пјҢеҲҷж— и®әиҜҘиҠӮзӮ№еҰӮдҪ•ж”№еҸҳпјҢReact йғҪдјҡе°Ҷ ref еҜ№иұЎзҡ„ .current еұһжҖ§и®ҫзҪ®дёәзӣёеә”зҡ„ DOM иҠӮзӮ№
    • useRef() пјҡеҸҜд»ҘеҫҲж–№дҫҝең°дҝқеӯҳд»»дҪ•еҸҜеҸҳеҖјпјҢеӣ дёәе®ғеҲӣе»әзҡ„жҳҜдёҖдёӘжҷ®йҖҡJSеҜ№иұЎгҖӮиҖҢ useRef() е’ҢиҮӘе»әдёҖдёӘ {current: ...} еҜ№иұЎзҡ„е”ҜдёҖеҢәеҲ«жҳҜпјҢuseRef дјҡеңЁжҜҸж¬ЎжёІжҹ“ж—¶иҝ”еӣһеҗҢдёҖдёӘ ref еҜ№иұЎгҖӮ
    • жіЁж„ҸпјҡеҪ“ ref еҜ№иұЎеҶ…е®№еҸҳеҢ–ж—¶пјҢuseRef е№¶дёҚдјҡйҖҡзҹҘдҪ гҖӮеҸҳжӣҙ .current еұһжҖ§дёҚдјҡеј•еҸ‘з»„д»¶йҮҚж–°жёІжҹ“гҖӮеҰӮжһңжғіиҰҒеңЁ React з»‘е®ҡжҲ–и§Јз»‘ DOM иҠӮзӮ№зҡ„ ref ж—¶иҝҗиЎҢжҹҗдәӣд»Јз ҒпјҢеҲҷйңҖиҰҒдҪҝз”Ёеӣһи°ғ ref жқҘе®һзҺ°гҖӮ

  • иҝҷйҮҢдёҚжҳҜеҫҲзҗҶи§Јпјҹпјҹпјҹ

    • еҮҪж•°з»„д»¶дёҚеҶҚеҢәеҲҶmountе’ҢupdateдёӨдёӘзҠ¶жҖҒпјҢжүҖд»ҘеҮҪж•°з»„д»¶жҜҸж¬Ўи°ғз”ЁйғҪдјҡжү§иЎҢеҶ…йғЁзҡ„жүҖжңүйҖ»иҫ‘пјҢжҖ§иғҪжҚҹиҖ—
  • зҲ¶з»„д»¶пјҡВ·
  • еӯҗз»„д»¶пјҡВ·
    • зӮ№еҮ»ж•°еӯ—пјҢи®Ўж—¶еҷЁдјҡжӣҙж–°пјҢдҪҶжҳҜеҗҚеӯ—зӣёе…ізҡ„еұ•зӨәз»„д»¶пјҢдёҺcountжҜ«ж— е…ізі»пјҢдҪҶжҳҜд№ҹеңЁжү§иЎҢпјҢиҝҷжҳҜеңЁеҒҡж— з”Ёзҡ„жёІжҹ“гҖӮ
    • жҲ‘д»¬еёҢжңӣпјҢеҸӘжңүдёҺе®ғзӣёе…ізҡ„жӣҙж–°дәҶпјҢжүҚжү§иЎҢеҜ№еә”зҡ„ж–№жі•

  • дјҳеҢ–пјҡВ·

    useMemoеҢ…иЈ№пјҢе№¶дј йҖ’з¬¬дәҢдёӘеҸӮж•°пјҢеҸӮж•°еҢ№й…ҚжҲҗеҠҹпјҢжүҚдјҡжү§иЎҢ

жіЁж„ҸпјҡВ·

дҫқиө–йЎ№ж•°з»„дёҚдјҡдҪңдёәеҸӮж•°дј вҖңеҲӣе»әвҖқеҮҪж•°гҖӮ

иҷҪз„¶д»ҺжҰӮеҝөдёҠжқҘиҜҙе®ғиЎЁзҺ°дёәпјҡжүҖжңүвҖңеҲӣе»әвҖқеҮҪж•°дёӯеј•з”Ёзҡ„еҖјйғҪеә”иҜҘеҮәзҺ°еңЁдҫқиө–йЎ№ж•°з»„дёӯгҖӮжңӘжқҘзј–иҜ‘еҷЁдјҡжӣҙеҠ жҷәиғҪпјҢеұҠж—¶иҮӘеҠЁеҲӣе»әж•°з»„е°ҶжҲҗдёәеҸҜиғҪгҖӮ

жҺЁиҚҗеҗҜз”Ё eslint-plugin-react-hooks дёӯзҡ„ exhaustive-deps и§„еҲҷгҖӮжӯӨи§„еҲҷдјҡеңЁж·»еҠ й”ҷиҜҜдҫқиө–ж—¶еҸ‘еҮәиӯҰе‘Ҡе№¶з»ҷеҮәдҝ®еӨҚе»әи®®гҖӮ

зү№жҖ§пјҡВ·

  • иҝ”еӣһдёҖдёӘ memoized еӣһи°ғеҮҪж•°

  • жҠҠеҶ…иҒ”еӣһи°ғеҮҪж•°еҸҠдҫқиө–йЎ№ж•°з»„дҪңдёәеҸӮж•°дј е…Ҙ useCallbackпјҢе®ғе°Ҷиҝ”еӣһиҜҘеӣһи°ғеҮҪж•°зҡ„ memoized зүҲжң¬пјҢиҜҘеӣһи°ғеҮҪж•°д»…еңЁжҹҗдёӘдҫқиө–йЎ№ж”№еҸҳж—¶жүҚдјҡжӣҙж–°гҖӮ

  • еҪ“дҪ жҠҠеӣһи°ғеҮҪж•°дј йҖ’з»ҷз»ҸиҝҮдјҳеҢ–зҡ„е№¶дҪҝз”Ёеј•з”ЁзӣёзӯүжҖ§еҺ»йҒҝе…Қйқһеҝ…иҰҒжёІжҹ“пјҲдҫӢеҰӮ shouldComponentUpdateпјүзҡ„еӯҗз»„д»¶ж—¶пјҢе®ғе°Ҷйқһеёёжңүз”ЁгҖӮ

    жҖҺд№ҲзҗҶи§ЈпјҹпјҹпјҹжҢ–еқ‘~

  • жҺЁиҚҗеҗҜз”Ё eslint-plugin-react-hooks дёӯзҡ„ exhaustive-deps и§„еҲҷгҖӮжӯӨи§„еҲҷдјҡеңЁж·»еҠ й”ҷиҜҜдҫқиө–ж—¶еҸ‘еҮәиӯҰе‘Ҡе№¶з»ҷеҮәдҝ®еӨҚе»әи®®

  • дёҺ useEffect дёҚеҗҢзҡ„жҳҜпјҢе®ғдјҡеңЁжүҖжңүзҡ„ DOM еҸҳжӣҙд№ӢеҗҺеҗҢжӯҘи°ғз”Ё effectпјҢеҸҜд»ҘдҪҝз”Ёе®ғжқҘиҜ»еҸ– DOM еёғеұҖе№¶еҗҢжӯҘи§ҰеҸ‘йҮҚжёІжҹ“гҖӮ
  • еңЁжөҸи§ҲеҷЁжү§иЎҢз»ҳеҲ¶д№ӢеүҚпјҢuseLayoutEffect еҶ…йғЁзҡ„жӣҙж–°и®ЎеҲ’е°Ҷиў«еҗҢжӯҘеҲ·ж–°гҖӮ

  • жҲ–жҳҜе°ҶиҜҘз»„д»¶е»¶иҝҹеҲ°е®ўжҲ·з«ҜжёІжҹ“е®ҢжҲҗеҗҺеҶҚжҳҫзӨәпјҲеҰӮжһңзӣҙеҲ° useLayoutEffect жү§иЎҢд№ӢеүҚ HTML йғҪжҳҫзӨәй”ҷд№ұзҡ„жғ…еҶөдёӢпјүгҖӮ
